# Vision

## Product thesis

Build a personal HQ that combines:

- an OS-style workspace interaction model
- keyboard-first navigation
- docs-quality information surfaces
- opinionated workflows for daily execution

## Who this is for

Initially: the builder themself.

Secondarily: a demo-quality platform showing product + engineering execution maturity.

## Success criteria for v1

- users can navigate core workflows without touching mouse frequently
- app context restores between sessions
- docs/info surfaces are visually high-quality and easy to scan
- architecture supports adding backend-powered features without rewrite

## What makes this differentiated

1. Keyboard-first operations as default, not add-on
2. Stateful workspace continuity
3. Premium docs UX built into core experience
4. Opinionated HQ modules rather than generic dashboards
